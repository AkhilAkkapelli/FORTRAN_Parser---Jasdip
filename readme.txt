1. Python version  - Python 3.7.3

2. Install requirements - 
   pip install -r requirements.txt

3. Run tests main_unparse.py ( converts fortran code to AST and AST to fortran code and generates symbol table from AST )
   sh ./run_tests_main_unparse_latex.sh
 
4. Run tests main_unparse_go.py ( converts fortran code to AST and AST to go code using symbol table generated from AST )
   sh ./run_tests_main_unparse_go.sh
